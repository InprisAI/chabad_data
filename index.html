Data Files

Master Index Quick Reference Full JSON (12.22 MB)

锔 Warning: The full JSON file is 12.22 MB and may be too large for some LLMs. Use the split chunk files instead!

Search Capabilities

Each document includes:

  • Metadata: Hebrew dates, Gregorian dates, author, opening phrases
  • Topics: Hebrew concepts (砖, 爪拽, 转转, etc.)
  • Key Concepts: English translations (Divine Presence, Righteous ones, Lower worlds)
  • References: Biblical, Talmudic, and Chassidic citations
  • Chunked Text: Main text divided into 1,000-1,500 character searchable chunks
  • Footnotes & Glossary: Complete annotations and definitions

Collection Structure

The JSON file contains:

  • collection_metadata - Overview and statistics
  • documents - Array of 195 documents, each with:
    • id - Document identifier (e.g., "1_")
    • reference - Full reference string
    • metadata - Dates, topics, concepts, references
    • content - Main text, chunks, footnotes, glossary
    • search_tags - Keywords for search

Usage Instructions for LLMs (Grok, Perplexity, etc.)

Method 1: Using Split Chunk Files (Recommended)

  1. Step 1 - Check Master Index: Query the master index to find which chunks contain your topic: https://raw.githubusercontent.com/nisyron/chabad_data/main/maamarim_master_index.json
    • Check topic_to_chunks for Hebrew topics (e.g., "砖")
    • Check concept_to_chunks for English concepts (e.g., "Divine Presence")
    • Check date_to_chunks for dates
    • Check doc_id_to_chunk for specific document IDs
  2. Step 2 - Search Chunk Files: Once you know which chunks (e.g., chunks 1, 2, 3), search those specific files: https://raw.githubusercontent.com/nisyron/chabad_data/main/maamarim_chunks/maamarim_chunk_01.json
    • Each chunk file is ~0.6 MB (much easier for LLMs to parse)
    • Each chunk contains 10 documents
    • There are 20 chunk files total

Method 2: Using Individual Document Files

For a specific document, use the individual file:

https://raw.githubusercontent.com/nisyron/chabad_data/main/maamarim_docs/maamarim_doc_1_.json

Method 3: Using Full File (Not Recommended)

The full file is 12.22 MB and may exceed LLM token limits:

https://raw.githubusercontent.com/nisyron/chabad_data/main/maamarim_structured.json

File Structure

  • maamarim_master_index.json (0.04 MB) - Master index mapping topics/concepts to chunk files
  • maamarim_chunks/ - 20 chunk files (~0.6 MB each, 10 documents per chunk)
  • maamarim_docs/ - 195 individual document files (one per document)
  • maamarim_structured.json (12.22 MB) - Complete file (may be too large for some LLMs)
  • maamarim_quick_reference.txt - Human-readable overview
